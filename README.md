# 小兔鲜儿电商项目

> 这是一个仓库迁移的项目，之后会继续开发。
>
> 为什么迁移呢？因为vue cli3构建速度太慢了，开发体验极差，所以脚手架迁移到vite
>
> 当然没有在原仓库上迁移脚手架，怕搞没，所以才有了这个项目

* 技术栈：
  * CSS预处理器：less
  * 包管理工具：pnpm
  * 脚手架：vite4
  * 框架：vue3
  * UI组件库：ant-design-vue
  * 图标库：@ant-design-vue 和 @icon-park/next
  * 第三方工具：
    * vuex
    * vue-router
    * @vueuse
    * axios
    * Eslint
